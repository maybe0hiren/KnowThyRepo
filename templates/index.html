KnowThyRepo

Backend API Open To All

JavaScript API

await fetch("/indexRepo", {
  method:"POST",
  headers:{
    "Content-Type":"application/json"
  },
  body:JSON.stringify({
    repoLink:"YOUR_REPO"
  })
});

const response = await fetch("/ask",{
  method:"POST",
  headers:{
    "Content-Type":"application/json",
    "Authorization":"Bearer YOUR_GEMINI_KEY"
  },
  body:JSON.stringify({
    repoLink:"YOUR_REPO",
    question:"How does authentication work?"
  })
});

console.log(await response.json());

Flask API

import requests

requests.post(
    "/indexRepo",
    json={
        "repoLink":"YOUR_REPO"
    }
)

response = requests.post(
    "/ask",
    headers={
        "Authorization":"Bearer YOUR_GEMINI_KEY"
    },
    json={
        "repoLink":"YOUR_REPO",
        "question":"How does authentication work?"
    }
)

print(response.json())
KnowThyRepo is open source. Gemini API keys are never stored, logged, or persisted by this service.
